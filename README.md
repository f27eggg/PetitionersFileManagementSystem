# 上访人员重点监控信息管理系统

## 项目简介

**济南市公安局历下分局刑侦大队 - 上访人员重点监控信息管理系统**

本系统是一个用于管理上访人员档案信息的桌面应用程序（exe），旨在帮助管理人员高效地上传、查询、管理上访人员的相关信息（包括照片管理），为后续上访人员的管控工作提供参考依据。系统采用现代化科技感UI设计，支持数据可视化地图展示。

## 系统特点

- **高度可移植性**：无需额外安装运行环境，任意Windows电脑均可直接运行，数据和照片采用相对路径存储
- **科技感UI界面**：深色主题、霓虹色高光、流畅动画、现代化专业设计
- **轻量级存储**：采用JSON文件+照片文件夹存储方案，无需配置数据库
- **操作简便**：提供直观的增删改查功能，支持拖拽上传照片
- **可视化地图**：中国省级行政区划地图展示人员分布统计

## 核心功能

1. **信息展示**：以表格和详情页形式展示上访人员的完整档案信息（含照片轮播）
2. **信息查询**：支持多维度条件检索（姓名、身份证号、危险等级等）
3. **信息录入**：提供表单界面新增上访人员档案，支持批量上传照片
4. **信息修改**：支持对现有档案信息进行编辑更新，管理照片（上传/删除/排序）
5. **统计可视化**：中国地图交互式展示 + 多维度统计图表（危险等级、上访次数、籍贯、进京方式）
6. **照片管理**：支持上访人员和关系人的多张照片上传、查看、删除
7. **数据导出**：默认导出全部数据为Excel/CSV格式，可选包含照片文件

## 数据结构

系统管理的上访人员档案包含以下四大模块：

| 模块 | 说明 | 字段数 |
|------|------|--------|
| 个人信息 | 基本身份、联系方式（动态列表）、上访记录、照片等 | 17 |
| 在京关系人 | 在北京的联系人信息、照片 | 6 |
| 信访案件 | 诉求、解决方式、进京方式等 | 7 |
| 评估结果 | 危险等级 | 1 |

**总计：31个字段**（v1.1版本调整）

详细字段说明请参阅 [数据结构设计文档](docs/数据结构设计.md)

## 技术栈

- **开发语言**：Java 17 (JDK 17.0.12)
- **UI框架**：JavaFX 17.0.2
- **数据存储**：JSON文件（Jackson 2.16.0）
- **构建工具**：Maven 3.9+
- **打包方式**：jpackage（内嵌JRE）
- **Excel处理**：Apache POI 5.2.5
- **日志框架**：Logback 1.4.14
- **地图可视化**：SVG + JavaFX
- **SVG支持**：Apache Batik 1.17（可选）
