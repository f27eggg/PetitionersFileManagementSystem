# 上访人员重点监控信息管理系统

## 项目简介

本系统是一个用于管理上访人员档案信息的桌面应用程序（exe），旨在帮助管理人员高效地上传、查询、管理上访人员的相关信息，为后续上访人员的管控工作提供参考依据。

## 系统特点

- **高度可移植性**：无需额外安装运行环境，任意Windows电脑均可直接运行
- **科技感UI界面**：现代化、美观、专业的用户界面设计
- **轻量级存储**：采用文件存储方案，无需配置数据库
- **操作简便**：提供直观的增删改查功能

## 核心功能

1. **信息展示**：以表格和详情页形式展示上访人员的完整档案信息
2. **信息查询**：支持多维度条件检索（姓名、身份证号、危险等级等）
3. **信息录入**：提供表单界面新增上访人员档案
4. **信息修改**：支持对现有档案信息进行编辑更新
5. **统计分析**：可视化图表展示各类统计数据

## 数据结构

系统管理的上访人员档案包含以下四大模块：

| 模块 | 说明 | 字段数 |
|------|------|--------|
| 个人信息 | 基本身份、联系方式、上访记录等 | 19 |
| 在京关系人 | 在北京的联系人信息 | 5 |
| 信访案件 | 诉求、解决方式、进京方式等 | 7 |
| 评估结果 | 统计分值、危险等级 | 2 |

详细字段说明请参阅 [数据结构设计文档](docs/数据结构设计.md)

## 技术栈

- **开发语言**：Java 17 (JDK 17.0.12)
- **UI框架**：JavaFX
- **数据存储**：JSON文件
- **构建工具**：Maven
- **打包方式**：jpackage
