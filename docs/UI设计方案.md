## 一、设计理念

### 1.1 核心设计原则

**科技感 + 专业性 + 易用性**

- **科技感**：深色主题、霓虹色高光、扁平化图标、流畅动画
- **专业性**：信息层次清晰、数据展示准确、配色稳重大气
- **易用性**：操作简单直观、响应及时、错误提示友好

### 1.2 设计关键词

```
深邃 | 锐利 | 未来 | 精准 | 高效 | 沉浸
```

## 五、组件设计规范

### 5.1 按钮设计

#### 主要按钮（Primary）

```css
背景：电子蓝渐变 (linear-gradient #58A6FF → #1F6FEB)
文字：纯白 #FFFFFF
圆角：6px
内边距：10px 20px
悬停：亮度提升 + 轻微放大(1.05)
点击：按下效果 scale(0.95)
```

#### 次要按钮（Secondary）

```css
背景：透明
边框：1px 电子蓝
文字：电子蓝
悬停：电子蓝半透明背景
```

#### 危险按钮（Danger）

```css
背景：激光红渐变
文字：纯白
悬停：红色加深
```

### 5.2 卡片设计

```css
背景：深墨蓝 #161B22
边框：1px 暗青灰 #30363D
圆角：8px
内边距：20px
阴影：0 4px 12px rgba(0,0,0,0.3)
悬停：轻微上浮 translateY(-4px) + 蓝色发光阴影
```

### 5.3 输入框设计

```css
背景：深墨蓝 #161B22
边框：1px 暗青灰 #30363D
圆角：4px
内边距：8px 12px
文字：白色 #C9D1D9
占位符：深灰 #8B949E
聚焦：边框变电子蓝 + 发光效果 box-shadow: 0 0 0 3px rgba(88,166,255,0.3)
```

### 5.4 图标规范

- **尺寸**：16px / 20px / 24px / 32px（统一倍数）
- **样式**：线性图标（stroke）、粗细2px
- **颜色**：继承父元素文字颜色
- **来源**：Feather Icons / Lucide Icons

## 六、动画与交互

### 6.1 页面切换动画

```
淡入淡出（Fade）+ 轻微位移（10px）
时长：300ms
缓动：ease-in-out
```

### 6.2 悬停反馈

- **按钮**：颜色加亮 + 轻微放大
- **卡片**：上浮 + 发光阴影
- **表格行**：背景色变化 + 左侧竖线
- **图标**：颜色变化 + 旋转（如刷新图标）

### 6.3 加载动画

```
旋转圆环（Spinner）
颜色：电子蓝
位置：内容区域中央
遮罩：半透明黑色背景
```

### 6.4 消息提示（Toast）

```
位置：窗口右上角
样式：卡片 + 对应颜色左边框
动画：从右侧滑入
自动关闭：3秒
```

| 类型 | 左边框颜色 | 图标 |
|------|-----------|------|
| 成功 | 矩阵绿 | ✅ |
| 警告 | 警戒橙 | ⚠️ |
| 错误 | 激光红 | ❌ |
| 信息 | 电子蓝 | ℹ️ |

## 八、响应式设计

### 8.1 最小分辨率

```
最小支持：1280 x 800
推荐分辨率：1920 x 1080
```

### 8.2 自适应规则

- **侧边栏**：固定宽度200px，可折叠为图标模式（60px）
- **内容区**：弹性宽度，随窗口自适应
- **表格**：超宽时水平滚动，保持列宽合理
- **图表**：等比缩放，保持纵横比

## 九、可访问性（Accessibility）

### 9.1 对比度要求

- **正常文字**：至少4.5:1（符合WCAG AA级）
- **大号文字**：至少3:1
- **当前配色对比度检查**：✅ 均符合标准

### 9.2 键盘导航

- 支持Tab键切换焦点
- 聚焦元素明显高亮
- 支持Enter/Space激活按钮

## 十二、开发注意事项

### 12.1 JavaFX CSS实现

JavaFX支持CSS样式，但语法略有不同：

```css
/* JavaFX CSS示例 */
.button-primary {
    -fx-background-color: linear-gradient(#58A6FF, #1F6FEB);
    -fx-text-fill: white;
    -fx-background-radius: 6;
    -fx-padding: 10 20 10 20;
}

.button-primary:hover {
    -fx-background-color: linear-gradient(#79B8FF, #388BFD);
    -fx-scale-x: 1.05;
    -fx-scale-y: 1.05;
}
```

### 12.2 图标使用

推荐使用SVG图标或IconFont：
- 引入FontAwesome字体
- 或使用JavaFX的SVGPath
