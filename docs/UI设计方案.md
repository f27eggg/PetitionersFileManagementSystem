## 一、设计理念

### 1.1 核心设计原则

**科技感 + 专业性 + 易用性**

- **科技感**：深色主题、霓虹色高光、扁平化图标、流畅动画
- **专业性**：信息层次清晰、数据展示准确、配色稳重大气
- **易用性**：操作简单直观、响应及时、错误提示友好

### 1.2 设计关键词

```
深邃 | 锐利 | 未来 | 精准 | 高效 | 沉浸
```

## 二、配色方案

### 2.1 主色调 - 深空蓝黑系

#### 背景色系

| 用途 | 颜色 | 十六进制 | 说明 |
|------|------|----------|------|
| **主背景** | 深空黑 | `#0D1117` | 最深背景，主窗口底色 |
| **次级背景** | 深墨蓝 | `#161B22` | 卡片、面板背景 |
| **悬浮背景** | 钢铁蓝 | `#1C2128` | 鼠标悬停时的背景色 |
| **边框色** | 暗青灰 | `#30363D` | 分隔线、边框 |

#### 前景色系

| 用途 | 颜色 | 十六进制 | 说明 |
|------|------|----------|------|

**收起状态（60px宽）**：
```
┌─────┐
│  ☰  │ ← 展开按钮
├─────┤
│  ⚡ │ ← 仅图标
├─────┤
│  📋 │
├─────┤
│  🔍 │
├─────┤
│  📊 │
├─────┤
│  📥 │
├─────┤
│  ⚙️ │
└─────┘
```

#### 伸缩交互设计

**触发方式**：
- 点击顶部折叠/展开按钮
- 鼠标悬停导航栏时显示提示文字（收起状态下）

**动画效果**：
- 宽度变化：200px ↔ 60px
- 动画时长：300ms
- 缓动函数：ease-in-out
- 内容区域同步调整宽度

**状态记忆**：
- 保存用户偏好设置
- 下次启动时恢复上次状态

#### 导航样式
- **默认状态**：半透明灰底 + 浅灰文字
- **悬停状态**：蓝色左边框（3px） + 加亮背景 + 显示工具提示
- **选中状态**：电子蓝左边框（3px） + 蓝色背景渐变 + 白色文字
- **图标**：统一使用24x24px图标
- **收起时悬停**：显示浮动提示框（显示菜单名称）

#### 中国地图可视化设计

**地图技术方案**：
- 使用SVG格式的中国省级行政区划地图
- 34个省级行政区（含港澳台）
- 地图尺寸：约700x600px（自适应容器）

**配色分级方案**：

根据选择的统计口径，地图省份按数量分为5个等级：

| 数量区间 | 颜色 | 十六进制 | 说明 |
|---------|------|----------|------|
| 0人 | 深灰 | `#30363D` | 无数据省份 |
| 1-5人 | 浅蓝 | `#388BFD` | 低密度 |
| 6-10人 | 中蓝 | `#58A6FF` | 中等密度 |
| 11-20人 | 深蓝 | `#1F6FEB` | 较高密度 |
| 20+人 | 电子蓝高亮 | `#79B8FF` | 高密度，发光效果 |

**交互功能**：

1. **悬停效果**：
   - 省份边界高亮（电子蓝，2px）
   - 显示浮动提示框：
     ```
     ┌─────────────────┐
     │ 山东省           │
     │ 总人数：23人     │
     │ 高危：5人        │
     │ 占比：18.5%      │
     └─────────────────┘
     ```
   - 省份颜色加亮（brightness 1.2）

2. **点击交互**：
   - 钻取到该省人员列表
   - 地图缩放到省份特写（动画）
   - 面包屑导航：全国 > 山东省

3. **动画效果**：
   - 初次加载：省份逐个浮现（延迟递增）
   - 口径切换：颜色渐变过渡（300ms）
   - 数据变化：脉冲动画

**地图边界效果**：
- 省份边界：1px 暗青灰（#30363D）
- 海岸线：2px 电子蓝发光（#58A6FF）
- 国界线：3px 金色（#D29922），发光效果
