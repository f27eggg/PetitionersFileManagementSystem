# 图标资源说明

本目录用于存放应用程序的图标资源。

## 图标清单

### 应用程序图标
- **app-icon.ico** - Windows应用程序图标（256x256）
- **app-icon.png** - PNG格式应用图标（512x512）

### 系统Logo
- **logo.png** - 系统Logo（警徽图标）
- 当前使用项目根目录的 `警徽.png`

### 菜单图标
由于JavaFX支持Unicode表情符号，当前界面使用Unicode图标：
- 📊 仪表盘
- 👥 人员管理
- 🔍 高级查询
- 📈 统计分析
- 💾 数据管理
- ⚙️ 系统设置

## 图标来源建议

1. **警徽图标**：使用政府机关标准警徽图片
2. **应用图标**：基于警徽图标制作，添加系统标识
3. **菜单图标**：可使用Unicode emoji或SVG图标

## 设计规范

- 主色调：深蓝色 (#1e3a8a, #1e40af)
- 辅助色：灰蓝色 (#334155, #475569)
- 背景色：深色 (#0f172a, #1e293b)
- 高亮色：蓝色 (#3b82f6)

## 制作工具推荐

- **在线工具**：
  - IconScout (https://iconscout.com/)
  - Flaticon (https://www.flaticon.com/)
  - Font Awesome (https://fontawesome.com/)

- **设计软件**：
  - Adobe Illustrator
  - Figma
  - Inkscape (免费)

## 注意事项

1. 应用图标建议尺寸：256x256, 512x512
2. 图标格式：PNG/ICO/SVG
3. 透明背景：建议使用透明背景PNG
4. 授权：确保图标使用权限
